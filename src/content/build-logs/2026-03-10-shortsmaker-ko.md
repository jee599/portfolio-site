광고용 쇼츠 영상을 자동으로 만들어주는 파이프라인을 구축하기 시작했다. 스크립트 입력부터 영상 출력까지 한 번에 처리하는 게 목표다.

## 왜 만들었나

쇼츠 영상 하나를 수작업으로 만들면 편집만 30분 이상 걸린다. 광고 목적이라 비슷한 포맷의 영상을 대량으로 뽑아야 하는데, 이걸 매번 사람이 하기엔 비효율적이다.

> 반복 작업은 자동화해야 한다. 특히 포맷이 정해진 콘텐츠는 더더욱.

## 파이프라인 구조

1. **스크립트 입력** — 텍스트 원고를 받는다
2. **AI TTS** — 텍스트를 음성으로 변환
3. **FFmpeg 렌더링** — 배경 영상 + 자막 + 음성을 합성
4. **YouTube 업로드** — API로 자동 업로드

```
스크립트 → TTS 생성 → FFmpeg 합성 → YouTube 업로드
```

## 현재 진행 상황

- Python 기반으로 각 단계를 모듈화하는 중이다
- FFmpeg 명령어 조합이 핵심. 자막 위치, 폰트, 타이밍을 템플릿으로 관리한다
- YouTube Data API v3로 업로드 자동화 연동 예정
